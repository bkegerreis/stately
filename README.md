# Instructions
Set up and activate a conda environment with all dependencies, run the app, then naviagte to http://127.0.0.1:8050 to explore Stately
```sh
$ cd /path/to/stately
$ conda create --name stately --file package-list.txt
$ conda activate stately
$ python app2.py
```
